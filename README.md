# hosipitalManagement
**医院管理系统**


## 服务端使用node koa框架搭建

## 客户端使用Vue框架 



```
注：主要用于医院的运营端，仅对管理员开放，患者无权访问。
```

### 人员

超级管理员：具有添加管理员的权限（无法前台操作，只能由后台添加）

管理员：具有对医院信息进行管理的权限

### 功能模块

* 注册
* 登录
* 管理模块
  1. 权限管理（仅超级管理员可见）
  2. 医院信息管理
     * 病例管理
     * 医院介绍
     * 科室管理
     * 广告管理
  3. 人员管理
     * 医生管理
     * 病人管理
  4. 对账管理
  5. 订单管理
     * 挂号管理
     * 问诊单管理
  6. 评价管理
